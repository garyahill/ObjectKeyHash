ObjectKeyHash

Summary

The code included in this project creates a data structure that associates pairs of objects and stores the pairs in a "hash-like" collection. Either object can then be used as the "index" (with the getAssociatedObject method) to return the other object. The clear all method removes all object pairs from the internal collection.

  • objectHashKey.js: contains the code for creating new instances of the data structure
  • usage.js: contains example code using an instance of the data structure
